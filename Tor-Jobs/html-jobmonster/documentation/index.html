HTML STRUCTURE

2.1. Basic Structure

The Basic Structure contains:

<div class="site">

<header class="noo-header" id="noo-header"></header> - top header, navbar toggle, main navigation

<div class=" container-wrap"></div> - all the content

<footer class=" colophon site-info"></footer> - footer colophon, footer navigation, footer-copyright

</div>

STYLESHEETS

3.1. Files

All CSS are located in the "css" folder of the template. Each CSS File is well commented and easy to use and customize to your own needs. Here's the list of the CSS files:

  1. font-awesome.min.css – FontAwesome.
  2. chosen.css
  3. custom.css
  4. jquery.datetimepicker.css
  5. style.css

Css which is installed skin color, that is "css/color" file

  1. default.css
  2. blue.css
  3. cyan.css
  4. pink.css

Note: font-awesome.min.css,style.css must declare on all pages of JobMonster pages.

SITE LAYOUT

5.1. Wide Layout

Wide layout is default layout.

 

5.2. Boxed Layout

You need to add class boxed-layoutin each body tag.

HTML

<body class="boxed-layout">

{{BODY CONTENT}}

</body>

 

CSS – STYLE.CSS

body.boxed-layout {

        background-color: #fff;

        background-image: url('../images/patterns/random_grey_variations.png');

        background-repeat: repeat;

}

body.boxed-layout .site {

        position: relative;

        overflow: hidden;

        background: #fff;

        -webkit-box-shadow: 0 0 7px rgba(0, 0, 0, 0.4);

        box-shadow: 0 0 7px rgba(0, 0, 0, 0.4);

        padding: 0;

        margin: 0 auto;

}

@media screen and (min-width: 768px) {

        body.boxed-layout .site, body.boxed-layout .noo-header {

                width: 750px;

        }

}

@media screen and (min-width: 992px) {

        body.boxed-layout .site, body.boxed-layout .noo-header {

                width: 970px;

        }

}

@media screen and (min-width: 1200px) {

        body.boxed-layout .site, body.boxed-layout .noo-header {

                width: 1200px;

        }

}

COLORS

CSS for skin color of the theme is located in folder “/css/color”:

  1. default.css
  2. blue.css
  3. cyan.css
  4. pink.css

HTML

<link rel="stylesheet" href="css/color/default.css">

 

Default style is file default.css. To select other colors, file default.css will be replaced with file blue.css, cyan.css, pink.css or orange.csscorresponding to the color you want.

Job Map

HTML

<div class="job-map">
    <div class="gmap-loading">Loading Maps 
        <div class="gmap-loader">
            <div class="rect1"></div>
            <div class="rect2"></div>
            <div class="rect3"></div>
            <div class="rect4"></div>
            <div class="rect5"></div>
        </div>
    </div>
    <div id="gmap" data-latitude="40.714398" data-longitude="-74.005279"></div>
</div>

Default coordinates is -74.005279(longitude) and 40.714398(latitude)

You can change the locations of Job Marker on the map by editing data-nooGmapL10n.js file
